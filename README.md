# JavaDSA 🚀
- A collection of **Data Structures and Algorithms (DSA)** implemented in **Java** for practice and learning purposes.  
- This repository is **ongoing** and will be updated regularly as I add more problems and solutions.  

## 📂 Folder Structure
- The repo is organized topic-wise for easy navigation:
  - **ArrayList** – Implementations and problems using ArrayList
  - **Arrays** – Array-based problems and techniques
  - **BST (Binary Search Trees)** – BST implementation and problems
  - **Backtracking** – Classic backtracking problems
  - **Basic Java** – Core Java basics used in DSA
  - **Binary Trees** – Binary tree creation and traversal problems
  - **Graphs** – Graph algorithms (BFS, DFS, etc.)
  - **Greedy Algorithms** – Greedy strategy-based problems
  - **Hashing** – HashMap, HashSet, and hashing-based problems
  - **Heaps** – Min-heap, max-heap, and heap problems
  - **Linked Lists** – Singly & doubly linked list implementations
  - **OOPS** – Object Oriented Programming concepts in Java
  - **Practice** – Miscellaneous DSA practice questions
  - **Queue** – Queue and circular queue implementations
  - **Recursion** – Recursive solutions to problems
  - **Stacks** – Stack implementations and problems
  - **Strings** – String manipulation problems
  - **Tries** – Trie (prefix tree) implementation

## 🛠️ Tech Stack
- **Language:** Java ☕  
- **IDE:** IntelliJ IDEA / VS Code / Eclipse  

## 📖 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/sahitya1903/JavaDSA.git
    ```
2. Open it in your preferred IDE.
- Navigate to the topic folder (e.g., Arrays, Graphs) and run the .java files.
